# Handmade Flowers E-Commerce Project

A complete, production-ready MERN (MongoDB, Express, React, Node.js) e-commerce website for handmade artificial pipe cleaner flowers.

## 📁 What's Included

### ✅ Complete Frontend (React)
- 8 customer-facing pages
- 4 admin management pages
- Responsive mobile design
- Pastel color theme
- All components & services

### ✅ Complete Backend (Node.js)
- RESTful API for all features
- Authentication & Authorization
- Database models & schemas
- File upload with Cloudinary
- Error handling middleware

### ✅ Database (MongoDB)
- Product schema
- Order schema
- Custom order schema
- Admin schema

### ✅ Features
- UPI QR code payment system
- Payment screenshot upload
- Order verification workflow
- Product management
- Custom order requests
- Real-time order status
- Admin dashboard

## 🚀 Quick Start

1. **Read QUICK_START.md** - Follow the 5-minute setup
2. **Setup MongoDB Atlas** - Free cloud database
3. **Setup Cloudinary** - Free image hosting
4. **Start Backend** - `npm install && npm start`
5. **Start Frontend** - `npm install && npm start`
6. **Create Admin** - http://localhost:3000/admin/login
7. **Add Products** - Start selling!
